# Orikawa Indonesia Website

Specialist Scaling Chiller & Chemical Cleaning Indonesia

## Features

- **Frontend**: Professional company website with services, products, projects, and contact pages
- **Admin Panel**: Full CMS to manage posts, products, projects, and site settings
- **Image Upload**: Upload and manage images for products and projects
- **WhatsApp Integration**: Contact form sends messages directly to WhatsApp

## Pages

### Public Pages
- `/` - Homepage with services overview
- `/services` - Detailed services (Scaling Chiller, Cleaning Cooling Tower, Chemical Cleaning)
- `/products` - Chemical products catalog with filtering
- `/projects` - Project portfolio with photos
- `/about` - Company information
- `/contact` - Contact form with WhatsApp integration

### Admin Pages
- `/admin` - Login page
- `/admin/dashboard` - Dashboard with statistics
- `/admin/posts` - Manage blog posts/information
- `/admin/products` - Manage chemical products
- `/admin/projects` - Manage project portfolio
- `/admin/settings` - Site settings

## Deploy to Vercel

1. Go to [vercel.com](https://vercel.com)

3. Click Deploy
4. Done!

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Local JSON storage (no database needed)

## Local Development

```bash
npm install
npm run dev
```
